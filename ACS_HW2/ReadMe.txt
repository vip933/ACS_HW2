Проект содержит 2 приложения: ACS_mdleypunskiy_HW2 - реализация основного задания на языке c++ с применением C-style и testMake - автоматический генератор тестов для основной программы, написаный на языке c++ без применения C-style. 
Компиляция осуществляется так:
	1. Открываем терминал из папки проекта ACS_mdlypunskiy_HW2
	2. Вводим команду: <cd build> (для перемещения в папку build)
	3. Вводим команду: <cmake ..> (для захвата CMakeLists в папке с проектом)
	4. Вводим команду: <cmake --build .> (Для создания системных файлов в текущей папке)
	5. Вводим команду: <cd ..> (для перехода в папку с проектом)
	6. Проект скомпилированав. (Системные файлы находятся в папке build, исполняемый файл находится в папке bin)

Запуск осуществляется так: (на примере запуска 1-ого теста)
	1. Открываем терминал из папки проекта ACS_mdlypunskiy_HW2
	2. Вводим команду: <./bin/project -f iofiles/test1.txt iofiles/out.txt iofiles/out2.txt>
	3. Программа запустилась)
